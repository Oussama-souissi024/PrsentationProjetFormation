📖 Présentation de la formation

🎯 Objectifs de la formation

Cette formation en .NET 8 vise à construire une application Full Stack en ASP.NET Core MVC, structurée selon les principes de la Clean Architecture.

L'objectif pédagogique est double :

  • Comprendre comment construire une application web MVC complète (UI, authentification, CRUD, panier, commandes)
  • Apprendre à structurer le code pour le rendre maintenable, testable et évolutif (niveau "pro")

Cette formation est spécialement conçue pour les jeunes diplômés souhaitant se préparer aux entretiens techniques pour des postes de développeur .NET Junior.

📊 Compétences acquises - 100% couverture Junior .NET

Ce projet couvre 100% des compétences techniques requises pour un poste de Développeur .NET Junior :

  • C# / .NET 8 - Classes, interfaces, LINQ, async/await
  • ASP.NET Core MVC - Controllers, Views, Razor, Validation
  • Entity Framework Core - DbContext, Migrations, Relations
  • Clean Architecture - Séparation en couches, DI, SOLID
  • ASP.NET Core Identity - Auth, Rôles, [Authorize]
  • REST API & JWT - Web API, Swagger, Authentication

Taux de réussite estimé en entretien technique : 80-85% (90%+ avec préparation algo + SQL)

🛠️ Projets de formation

Projet principal : Application e-commerce Full Stack en ASP.NET Core MVC avec Clean Architecture

Extension avancée : Refactoring vers architecture API + Client MVC (découplé)

La partie front MVC (Razor) est construite de 0 pour assurer une compréhension totale des mécanismes (Controllers, Views, UI/UX, validation)

🏗️ Architecture enseignée

Clean Architecture - Le principe central : les règles métier ne dépendent pas de l'UI ni de la base de données

Avantages :

  • Couches indépendantes et réutilisables
  • Code maintenable et testable
  • Facilité de remplacement de couche (ex: MVC → API)
  • Évolutivité et flexibilité

Preuve par l'exemple : L'extension démontre que vous pouvez remplacer la couche Présentation MVC par une API sans modifier une seule ligne de code dans Core, Application ou Infrastructure.

📚 Technologies et compétences abordées

Core : .NET 8, C#, Clean Architecture, SOLID Principles

Backend : ASP.NET Core MVC, Entity Framework, SQL Server, Repository Pattern

Frontend : Razor Views, Controllers, UI/UX, Validation client/serveur

API : REST APIs, JWT Authentication, Swagger/OpenAPI

Outils : Git & GitHub, User Secrets, Dependency Injection, AutoMapper

✨ Fonctionnalités du projet e-commerce

Application couvrant un cycle e-commerce complet :

  • Catalogue : Voir produits, détails, gestion catégories/coupons (admin)
  • Panier & Commandes : Ajouter au panier, appliquer coupon, passer commande, historique
  • Authentification : Inscription, connexion, reset mot de passe, rôles Admin/User
  • Extension API : Backend REST avec JWT + Client MVC consommant l'API
